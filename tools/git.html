1.1 快速使用github

创建一个新的库

echo "# arvinbook" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin git@github.com:arvinxuxie/arvinbook.git
git push -u origin master

push一个本地存在的库

git remote add origin git@github.com:arvinxuxie/arvinbook.git
git push -u origin master
